## 一、技术规格速查

### 1.1 图片规格

| 项目 | 要求 |
|------|------|
| 格式 | **JPG** |
| 手机尺寸 | **1440 × 2560 px** |
| 折叠屏尺寸 | **2934 × 3306 px** |
| 大小限制 | **1M 以内** |
| 预览视频 | ≤10M，封面图 1080×1920 或 1080×2340 |

### 1.2 命名规范

- 不超过 **10 个中文字**（含标点、空格）
- 须符合"**信、达、雅**"，简洁文雅，不可有歧义
- **禁止**前后空格，中间最多一个空格
- **禁止**特殊符号、敏感词汇
- 不可与官方资源名称雷同
- 不可包含第三方广告信息、网址、产品名
- **禁止**功能描述词（充电、流体）、系统名（OS、Win、鸿蒙）
- 同类型名称不能仅以数字（XX1、XX2）或时间（正午、傍晚）区分

### 1.3 标签规范

分 **4 个维度** 勾选，不标注同义词、无意义标签、与画面无关的标签：
- **画面标签**：图片主体内容描述
- **颜色标签**：主色调
- **气氛标签**：画面传达的情绪氛围
- **风格标签**：艺术风格分类

### 1.4 内容图片上传规则

- 猜谜/emoji/成语/明星小时候/找不同/谐音梗/脑筋急转弯等使用**大面积画框修饰**的插画图片，每月上传比例暂调整为当月审核上限的 **5%**
- Emoji猜谜专栏送审数量占比不得高于当月送审总图数的 **10%**（2月起执行）

## 二、内容红线清单

### 2.1 政治/意识形态（一票否决）

- 不得含有意识形态元素
- 不得出现各国国旗、国徽、地图（含可分辨国家地区轮廓的地球设计）
- 小熊维尼动画形象不得出现
- 不得恶搞、篡改、歪曲历史人物和事件
- 不得出现公职人员形象（包括交警等）
- 国际局势紧张期间，涉及**日本元素**（动漫/影视/旅游/医疗等）谨慎上传
- "琉球"表述须删除"日本"字样，改为"南西诸岛"或直接删除
- 金色边瓶子 + "皇帝"文案组合需谨慎，避免恶意解读

### 2.2 色情/低俗

- 禁止淫秽色情、软色情、性暗示
- 禁止大面积肉色透视装、胸沟明显、露肚脐
- 禁止兜裆布等不雅着装
- 眉钉、舌钉、脐钉等非主流身体装饰不推
- 纹身禁止出现

### 2.3 暴力/恐怖/惊悚

- 禁止血腥画面、打架斗殴、骷髅、动物骨架/干尸
- 禁止画面氛围诡异惊悚（用户打开锁屏可能吓一跳）
- 禁止表情痛苦、引人不适的内容
- 画面过于压抑、看着心酸难受的不上

### 2.4 宗教迷信

- 禁止观音、佛像、阴阳八卦、占卜算命、风水罗盘
- 禁止符纸
- 禁止涉及死亡相关（遗作、爱宠离世变钻石等）
- 禁止迷信内容（招财猫、风水气场等）

### 2.5 品牌/商业

- 不得出现任何品牌 Logo、品牌名（含华为/HUAWEI/HONOR/荣耀）
- 不得出现系统标识（EMUI/HarmonyOS/鸿蒙）
- 无授权不得使用品牌车辆、品牌玩偶、品牌设备
- 禁止广告/软广，不得出现 APP 名称、具体品牌名
- 禁止出现真实公司名称、真实个人信息、手机号、发票等
- 避免出现第三方引擎字样（掌酷等）
- 禁止出现任何形式的金钱元素或货币符号

### 2.6 奥运/世界杯

- 公司营销要求，**避免出现奥运和世界杯内容**，包括"奥运""世界杯"字眼

### 2.7 飞机/航行安全

- 避免出现危害航行的内容（不论现实还是影视），如鸟撞飞机
- 避免飞机机身残破/坠机现场感

### 2.8 非主流婚恋观

- 不上架同性恋、未成年人恋爱、不伦恋、婚外情、一夜情、不婚不育思想等内容
- 相关动漫（涉及LGBTQ主题的）不推

### 2.9 明星相关

- 避免花边新闻、负面内容、卖惨
- 避免给明星打非"大众化"标签（容易引发粉丝投诉）
- 避免"阴阳怪气"式文案
- 避免出现"踩一捧一"的配图
- 避免涉及法制咖
- 未经官宣的信息不传
- 明星真实学校/经历要核实准确
- 游戏、影视、明星三类资源上架需统一评审

### 2.10 社会敏感

- 避免现实社会负面内容
- 避免反映其他国家负面内容
- 避免涉及社会经济现状（如油价上涨引发新增成本等）
- 避免暗讽品牌/行业
- 避免涉及宗教信仰、民族饮食习惯（如回民不吃猪肉）
- 避免出现国家"三有"保护动物的食用内容
- 避免存在争议且未经普遍证实的事件

### 2.11 更多禁止元素

- 禁止素人正脸图片
- 禁止引导操作的按钮、二维码、条形码、水印
- 禁止为图片增加边框
- 禁止动物形象作为广告画面主体
- 禁止画面出现真实手机号、联系人名称
- 禁止简介中出现福利、返现、赠送等诱导内容

## 三、低质量内容判定标准

### 3.1 技术质量

- 存在噪点、马赛克、清晰度低、高曝光等问题
- 图片比例失调、界面不完整
- 文字过小、展示不完整、影响排版
- 预览视频拉伸变形、画面跳动闪烁、首尾不衔接

### 3.2 构图/审美

- 阴阳脸、主体物没有美感
- 画面简陋、元素质感廉价、完成度低
- 背景显脏、海水显脏、色调灰暗
- 纹理感过重、画风老气、画风低幼
- 画面像"页游感"、人脸质感粗糙
- 玻璃痕迹像屏幕有划痕
- 人物主体密集且杂乱
- 背景过亮与人物分不开
- 秃头不美观
- 食物油腻感过重
- 图片下方有色块造成锁屏切割感

### 3.3 模板/排版

- 模板图片中红框标题文字需小于黄框版式文字
- 字太多太密、费眼睛
- 同一模板未经二次设计不得直接用于多套资源
- 同一作品各元素风格应保持一致
- 不可使用"说明书式"预览图
- 不可制作"带手机"样式的封面图
- 模板落地页不能过于简单（如只写答案不解释梗）

### 3.4 传统/老气元素（禁止）

- 建筑、街景、书桌仅搭配昼夜切换
- 贴纸、对联、窗花、灯笼、烟花、鞭炮
- 流水、瀑布、凉亭、群山、荷塘、锦鲤、鹤、鹰等传统元素堆砌
- 蓝天、白云、草地、郁金香、彩霞等单调画面
- 海浪、海、沙滩、岩石等无功能性交互
- 星空、星云、星球、黑洞等合成图无创意动效
- 点线面抽象画面无功能性交互

## 四、版权规范

| 版权类型 | 需提交材料 |
|----------|-----------|
| **原创** | 原创声明 + 可分辨绘制过程的PSD源文件/截图/录屏 |
| **个人授权** | 身份证复印件 + 授权书（手写签名/指纹/印章） |
| **摄影图** | 带参数的摄影源文件（png/jpg/jpeg/MP4） |
| **素材库版权** | 来源说明、网站链接、是否商用及范围的文档 |
| **AI素材（原创）** | AI素材声明 + AI原始图样 + **再创作程度>50%** 的再绘制过程文件 |
| **AI素材（授权）** | 授权证明 + AI素材声明 + AI原始图样 + 再创作程度>50%的再绘制过程文件 |

- 确保设计用图的版权、人物肖像权、字体版权
- 作品登记证书**无法证明**著作权归属
- 无授权不可使用品牌车辆、品牌玩偶等品牌实物
- 未授权的盲盒产品图有广告嫌疑
- 周大生等品牌首饰图上传前需确认可识别性，有法律风险

## 六、各品类内容公式（优秀案例关键词提炼）

> 来源：7期月报"优秀案例"板块中各品类的关键词标注

### 6.1 插画（CTR: 8.57%~16.71%）

- **核心关键词**: 互动猜谜、谐音梗、历史冷知识、育儿知识、搞笑图集、盘点
- **标题公式**: 问句/挑战式 — "猜猜TA是谁?" "你能秒懂吗?" "xxx大挑战!"
- **落地页套路**: 给出谜面→引导思考→揭晓答案→解释原因/冷知识
- **高效内容类型**:
  - Emoji猜谜（CTR极高，可达19%+）
  - 谐音梗挑战（如"家电谐音梗" CTR可达15.9%）
  - 历史冷知识（"李白钱从哪来" "古人近视怎么办"）
  - 育儿知识插画（"孩子拖拉磨蹭怎么办"）
  - 美食猜谜（"美食猜谜大作战" CTR可达19.8%）
- **注意**: 大面积画框修饰的插画有5%总量限制

### 6.2 美食（CTR: 7.26%~9.63%）

- **核心关键词**: 猎奇、盘点、冷知识、搞笑图集
- **标题公式**: "盘点xxx" "xxx的真相" "别xxx" "xxx冷知识"
- **落地页套路**: 列出N个反常识事实→逐一解释→生活实用收尾
- **高效内容类型**:
  - 315曝光/避坑类（时效性强）
  - 毁牙/毁健康的食物
  - 难吃水果/难剥水果盘点
  - 食物冷知识（蜂蜜结晶、甜甜圈的洞）
  - 烘焙翻车/倒霉瞬间搞笑合集
  - 奶茶小料/零食避雷

### 6.3 动物（CTR: 3.77%~4.52%）

- **核心关键词**: 搞笑图集、猎奇、萌系、盘点
- **标题公式**: "xxx名场面" "xxx瞬间" "盘点xxx"
- **落地页套路**: 萌/搞笑图片集合→每张配一句话解读→轻松收尾
- **高效内容类型**:
  - 猫咪/狗狗的搞笑/翻车/拆家瞬间
  - 动物界的"神图"（长得像P过的）
  - 动物冷知识（长寿鲨鱼、长城猫）
  - 狗狗比丑大赛/表情包大赛
  - 动物"社死"瞬间
- **注意**: 萌动星球专栏要求"必须萌"

### 6.4 动漫（CTR: 3.92%~4.55%）

- **核心关键词**: 盘点、经典/热门IP、动漫新资讯、冷知识、彩蛋
- **标题公式**: "盘点xxx" "xxx的冷知识" "xxx放大看"
- **高效内容类型**:
  - 国漫/经典IP盘点（各省代表动漫、被认错的国漫）
  - 热门IP冷知识/彩蛋（熊出没、疯狂动物城）
  - 动漫新资讯（新电影/新番介绍）
  - 战力/角色排行
- **注意**: 动漫迷航专栏要求"必须是大电影"

### 6.5 影视（CTR: 2.88%~4.84%）

- **核心关键词**: 追剧清单、反差猎奇、经典回忆、热门影视资讯、盘点
- **标题公式**: "xxx追剧清单" "盘点xxx" "xxx天花板"
- **高效内容类型**:
  - 追剧/观影清单（季节性或年度）
  - 荧屏CP/婚纱照/名场面盘点
  - 猜剧名互动
  - 经典影视回忆杀
  - 待播剧前瞻/阵容介绍
  - 短剧新趋势（如3秒一集）
- **注意**: 需统一评审

### 6.6 明星（CTR: 3.77%~5.50%）

- **核心关键词**: 盘点、猎奇反差、明星美图、正面热点
- **标题公式**: "盘点xxx" "xxx的xxx" "原来xxx"
- **高效内容类型**:
  - 女星/男星造型盘点（古装、异瞳、皮衣等）
  - 明星冷知识（隐藏身份、本名揭秘、当过兵）
  - 素颜对比、撞脸盘点
  - 明星正面事件（演唱会、公益）
  - 春晚/节日相关动态
- **高风险区域**: 避免花边新闻、避免踩一捧一、避免"阴阳怪气"、避免涉及法制咖、核实学校/经历
- **内容禁区**: 避免品牌/商务/商业字眼（如"代言""商务女王""品牌大使"），避免以商务数据或商业成就作为选题角度
- **推荐调性**: 聚焦**轻科普**——明星本名/学历/经历等冷知识、服化道/拍摄/舞台等影视工业科普、红毯/戏服/台词等幕后文化知识、造型/气质/演技的审美分析。以"知识增量"为核心，而非商业话题或八卦

### 6.7 科技（CTR: 4.30%~4.74%）

- **核心关键词**: 猎奇、宇宙冷知识、外星生命
- **标题公式**: "xxx是福是祸?" "如果xxx会怎样?" "xxx的秘密"
- **高效内容类型**:
  - 宇宙/天文冷知识（黑洞、白洞、火星土壤）
  - 太空探索新发现
  - 星球对比/假设类（不同星球能活多久）
  - 未来科技展望
- **注意**: 航天内容避免涉及"面包被禁"等特定敏感话题

### 6.8 艺术（CTR: 2.95%~3.85%）

- **核心关键词**: 猎奇、搞笑图集、历史冷知识、精美文物鉴赏
- **标题公式**: "xxx的真相" "盘点xxx" "博物馆里的xxx"
- **高效内容类型**:
  - 文物冷知识（黄金能治病、蒙娜丽莎眉毛）
  - 博物馆"显眼包"（搞笑文物）
  - 古人生活/饮食趣闻（鲁迅年夜饭、皇帝吃饭）
  - 视错觉/奇观合集
  - 文物的神级撞脸
- **注意**: 馆藏经典专栏要求"一级博物馆馆藏"

### 6.9 旅行风光（CTR: 3.36%~4.21%）

- **核心关键词**: 猎奇、盘点
- **标题公式**: "盘点xxx" "xxxx真相" "xxx最后的xxx"

## 七、审核驳回高频雷区

> 来源：7期月报"审核驳回TOP问题"板块

### 7.1 图片问题 Top 雷区

| 雷区 | 具体表现 | 出现频率 |
|------|---------|---------|
| 不符合调性 | 氛围诡异/惊悚、画风低幼/老气、页游感、质感廉价 | ★★★★★ |
| 画面简陋 | 元素低质、细节粗糙、完成度低、像泡沫/堆砌 | ★★★★ |
| 人体/面部 | 阴阳脸、表情痛苦、露肚脐、胸沟明显、秃头 | ★★★★ |
| 敏感元素 | 枪、符纸、盲盒产品(广告嫌疑)、品牌Logo、国旗/地图 | ★★★★ |
| 模板问题 | 字太密、红框字号大于黄框、模板落地页太简单 | ★★★ |
| 背景问题 | 显脏、纹理过重、过亮与主体分不开、有切割感 | ★★★ |
| 非主流 | 眉钉/舌钉/脐钉、纹身、兜裆布 | ★★ |
| 素人/肖像 | 素人正脸、人物过于像人(诡异) | ★★ |

### 7.2 文案问题 Top 雷区

| 雷区 | 举例 | 出现频率 |
|------|------|---------|
| **错字/缺字** | "无滤镜"写成"无滤"、"蓝盈莹"写成"蓝盈盈"、"海葵"写成"海奎" | ★★★★★ |
| **有歧义** | "内娱活人天花板"不加引号、"面瘫"不被认为负面、"两幅面孔" | ★★★★ |
| **引号/书名号用反** | 双引号方向反、《这部剧》应为《剧名》 | ★★★ |
| **事实错误** | 演员和作品对不上、学校错误、"三代"写成"四代" | ★★★★ |
| **文案层次低** | "人不如狗"、隐含负面标签、违背公序良俗 | ★★★ |
| **过于夸张** | "命瞬间就没了"、换个头套就"地狱" | ★★ |
| **花边/八卦** | 明星恋情生日、涉及死亡/遗作 | ★★★ |

### 7.3 舆情/事实雷区

| 雷区 | 举例 |
|------|------|
| 演员-作品张冠李戴 | 王奕婷没演过《春闺梦里人》、周也演程霜非章若楠 |
| 歌曲-曲目错误 | 单依纯唱的是《珠玉》非《给电影人的情书》 |
| 化学式配错 | 文案说"氮气"，图配的是"氨气"化学式 |
| 动物-图不对应 | 写石斑鱼配翻车鲀、写绵羊配山羊 |
| 国籍/出生地错误 | 靳东山东非上海、朱一龙湖北非上海 |
| 概念混淆 | "尘卷风"与"雷暴大风"形成原因不同 |
| 时效过期 | 电视剧完结很久仍用"即将"、"2025年"内容在2026年仍用 |
| 未官宣信息 | 张艺兴演唱会未官宣、鹿晗结婚生子未证实 |
