## 快速安装

### Proma 工作区（推荐）

将本目录下文件复制到你的 Proma 工作区：

```
skills/huabao-content-planner/SKILL.md
workspace-files/.context/huabao-knowledge-base.md
```

重启会话后生效。

### Claude Code

将本目录放到 `~/.claude/skills/huabao-content-planner/`，Claude Code 启动后自动识别。

### 通用 AI 工具（GPT / Dify / Coze / 豆包 / Kimi 等）

打开 `standalone-prompt.md` → 全选复制 → 粘贴到系统提示词 / Instructions 中即可。

## 核心能力

### 策略反推

每次策划前先拆解月报优秀案例：标题用了什么句式？副标题的钩子在哪？图片选择了什么主体和氛围？抽取出可复用的策略模式。

### 标题六种高效模式

| 模式 | 特征 | 示例 |
|------|------|------|
| 问句悬念 | 抛出让人想知道答案的问题 | "你能看到什么形状？" |
| 猎奇反常识 | 打破日常认知的事实 | "长得像AI的动物" |
| 数字盘点 | 具体数字传递信息密度 | "全球五大奇险建筑" |
| 警示别xxx | "千万别"触发自我保护点击 | "千万别这样安慰人" |
| 引号梗 | 引号制造网感和默契 | ""吃同事"的大鱼" |
| 身份代入 | 让标题与"你"产生关联 | "学霸是这样养成的" |

### 副标题四法则

1. **接力不重复** — 标题抛悬念，副标题给具体事实钩子
2. **口语化** — 像朋友聊天，大量使用"？""！"
3. **"你"字高频** — 把读者拉入对话
4. **句末留余味** — 结尾有让人想点进去的冲动

### 红线自检+驳回对照

涵盖政治/色情/暴力/宗教/品牌/奥运/明星舆情等 30+ 条红线，对照月报真实驳回案例主动规避。

### 批量选题生成

每个内容包包含：标题（≤10字）+ 副标题（≤25字）+ 落地页文案 + 策略反推 + 红线自检。
