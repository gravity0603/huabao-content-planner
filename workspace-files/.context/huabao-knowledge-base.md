## 一、技术规格速查

### 1.1 图片规格

| 项目 | 要求 |
|------|------|
| 格式 | **JPG** |
| 手机尺寸 | **1440 × 2560 px** |
| 折叠屏尺寸 | **2934 × 3306 px** |
| 大小限制 | **1M 以内** |
| 预览视频 | ≤10M，封面图 1080×1920 或 1080×2340 |

### 1.2 命名规范

- 不超过 **10 个中文字**（含标点、空格）
- 须符合"**信、达、雅**"，简洁文雅，不可有歧义
- **禁止**前后空格，中间最多一个空格
- **禁止**特殊符号、敏感词汇
- 不可与官方资源名称雷同
- 不可包含第三方广告信息、网址、产品名
- **禁止**功能描述词（充电、流体）、系统名（OS、Win、鸿蒙）
- 同类型名称不能仅以数字（XX1、XX2）或时间（正午、傍晚）区分

### 1.3 标签规范

分 **4 个维度** 勾选，不标注同义词、无意义标签、与画面无关的标签：
- **画面标签**：图片主体内容描述
- **颜色标签**：主色调
- **气氛标签**：画面传达的情绪氛围
- **风格标签**：艺术风格分类

### 1.4 内容图片上传规则

- 猜谜/emoji/成语/明星小时候/找不同/谐音梗/脑筋急转弯等使用**大面积画框修饰**的插画图片，每月上传比例暂调整为当月审核上限的 **5%**
- Emoji猜谜专栏送审数量占比不得高于当月送审总图数的 **10%**（2月起执行）

## 二、内容红线清单

### 2.1 政治/意识形态（一票否决）

- 不得含有意识形态元素
- 不得出现各国国旗、国徽、地图（含可分辨国家地区轮廓的地球设计）
- 小熊维尼动画形象不得出现
- 不得恶搞、篡改、歪曲历史人物和事件
- 不得出现公职人员形象（包括交警等）
- 国际局势紧张期间，涉及**日本元素**（动漫/影视/旅游/医疗等）谨慎上传
- "琉球"表述须删除"日本"字样，改为"南西诸岛"或直接删除
- 金色边瓶子 + "皇帝"文案组合需谨慎，避免恶意解读

### 2.2 色情/低俗

- 禁止淫秽色情、软色情、性暗示
- 禁止大面积肉色透视装、胸沟明显、露肚脐
- 禁止兜裆布等不雅着装
- 眉钉、舌钉、脐钉等非主流身体装饰不推
- 纹身禁止出现

### 2.3 暴力/恐怖/惊悚

- 禁止血腥画面、打架斗殴、骷髅、动物骨架/干尸
- 禁止画面氛围诡异惊悚（用户打开锁屏可能吓一跳）
- 禁止表情痛苦、引人不适的内容
- 画面过于压抑、看着心酸难受的不上

### 2.4 宗教迷信

- 禁止观音、佛像、阴阳八卦、占卜算命、风水罗盘
- 禁止符纸
- 禁止涉及死亡相关（遗作、爱宠离世变钻石等）
- 禁止迷信内容（招财猫、风水气场等）

### 2.5 品牌/商业

- 不得出现任何品牌 Logo、品牌名（含华为/HUAWEI/HONOR/荣耀）
- 不得出现系统标识（EMUI/HarmonyOS/鸿蒙）
- 无授权不得使用品牌车辆、品牌玩偶、品牌设备
- 禁止广告/软广，不得出现 APP 名称、具体品牌名
- 禁止出现真实公司名称、真实个人信息、手机号、发票等
- 避免出现第三方引擎字样（掌酷等）
- 禁止出现任何形式的金钱元素或货币符号

### 2.6 奥运/世界杯

- 公司营销要求，**避免出现奥运和世界杯内容**，包括"奥运""世界杯"字眼

### 2.7 飞机/航行安全

- 避免出现危害航行的内容（不论现实还是影视），如鸟撞飞机
- 避免飞机机身残破/坠机现场感

### 2.8 非主流婚恋观

- 不上架同性恋、未成年人恋爱、不伦恋、婚外情、一夜情、不婚不育思想等内容
- 相关动漫（涉及LGBTQ主题的）不推

### 2.9 明星相关

- 避免花边新闻、负面内容、卖惨
- 避免给明星打非"大众化"标签（容易引发粉丝投诉）
- 避免"阴阳怪气"式文案
- 避免出现"踩一捧一"的配图
- 避免涉及法制咖
- 未经官宣的信息不传
- 明星真实学校/经历要核实准确
- 游戏、影视、明星三类资源上架需统一评审

### 2.10 社会敏感

- 避免现实社会负面内容
- 避免反映其他国家负面内容
- 避免涉及社会经济现状（如油价上涨引发新增成本等）
- 避免暗讽品牌/行业
- 避免涉及宗教信仰、民族饮食习惯（如回民不吃猪肉）
- 避免出现国家"三有"保护动物的食用内容
- 避免存在争议且未经普遍证实的事件

### 2.11 更多禁止元素

- 禁止素人正脸图片
- 禁止引导操作的按钮、二维码、条形码、水印
- 禁止为图片增加边框
- 禁止动物形象作为广告画面主体
- 禁止画面出现真实手机号、联系人名称
- 禁止简介中出现福利、返现、赠送等诱导内容

## 三、低质量内容判定标准

### 3.1 技术质量

- 存在噪点、马赛克、清晰度低、高曝光等问题
- 图片比例失调、界面不完整
- 文字过小、展示不完整、影响排版
- 预览视频拉伸变形、画面跳动闪烁、首尾不衔接

### 3.2 构图/审美

- 阴阳脸、主体物没有美感
- 画面简陋、元素质感廉价、完成度低
- 背景显脏、海水显脏、色调灰暗
- 纹理感过重、画风老气、画风低幼
- 画面像"页游感"、人脸质感粗糙
- 玻璃痕迹像屏幕有划痕
- 人物主体密集且杂乱
- 背景过亮与人物分不开
- 秃头不美观
- 食物油腻感过重
- 图片下方有色块造成锁屏切割感

### 3.3 模板/排版

- 模板图片中红框标题文字需小于黄框版式文字
- 字太多太密、费眼睛
- 同一模板未经二次设计不得直接用于多套资源
- 同一作品各元素风格应保持一致
- 不可使用"说明书式"预览图
- 不可制作"带手机"样式的封面图
- 模板落地页不能过于简单（如只写答案不解释梗）

### 3.4 传统/老气元素（禁止）

- 建筑、街景、书桌仅搭配昼夜切换
- 贴纸、对联、窗花、灯笼、烟花、鞭炮
- 流水、瀑布、凉亭、群山、荷塘、锦鲤、鹤、鹰等传统元素堆砌
- 蓝天、白云、草地、郁金香、彩霞等单调画面
- 海浪、海、沙滩、岩石等无功能性交互
- 星空、星云、星球、黑洞等合成图无创意动效
- 点线面抽象画面无功能性交互

## 四、版权规范

| 版权类型 | 需提交材料 |
|----------|-----------|
| **原创** | 原创声明 + 可分辨绘制过程的PSD源文件/截图/录屏 |
| **个人授权** | 身份证复印件 + 授权书（手写签名/指纹/印章） |
| **摄影图** | 带参数的摄影源文件（png/jpg/jpeg/MP4） |
| **素材库版权** | 来源说明、网站链接、是否商用及范围的文档 |
| **AI素材（原创）** | AI素材声明 + AI原始图样 + **再创作程度>50%** 的再绘制过程文件 |
| **AI素材（授权）** | 授权证明 + AI素材声明 + AI原始图样 + 再创作程度>50%的再绘制过程文件 |

- 确保设计用图的版权、人物肖像权、字体版权
- 作品登记证书**无法证明**著作权归属
- 无授权不可使用品牌车辆、品牌玩偶等品牌实物
- 未授权的盲盒产品图有广告嫌疑
- 周大生等品牌首饰图上传前需确认可识别性，有法律风险

## 六、各品类内容公式与选题策略

> 来源：7期月报优秀案例 + 5.21单品级CTR数据(6604条)

### 核心原则（数据驱动）

**"互动参与 > 被动浏览"**：5.21数据证明，让读者"参与进来"的互动型内容（猜谜、考考你、测测你）CTR比纯被动浏览的盘点型高出3-5倍。所有品类在构思标题时应优先考虑是否可加入互动元素。

---

### 6.1 插画（5.21加权CTR: 11.95% | 283条内容 | ⚠️供给不足）

- **核心关键词**: 互动猜谜、谐音梗、考考你、测测你
- **标题公式（数据验证后更新）**:
  - **#1 互动挑战型** — "xxx猜xxx""考考你的xxx""测测你的xxx"（数据支撑：谐音梗猜成语19.58%、考考你的小脑瓜14.62%、测测你的网感14.60%）
  - **#2 谐音梗/语言游戏型** — 利用语言双关制造"秒懂"或"想三秒才懂"的智力快感
  - #3 历史冷知识/育儿知识 — 传统稳定型，CTR中高但非顶尖
- **落地页套路**: 给出谜面/挑战→引导读者停下来思考→揭晓答案→解释背后的逻辑或冷知识
- **高效内容类型（按CTR排序）**:
