## 一、技术规格速查

### 1.1 图片规格

| 项目 | 要求 |
|------|------|
| 格式 | **JPG** |
| 手机尺寸 | **1440 × 2560 px** |
| 折叠屏尺寸 | **2934 × 3306 px** |
| 大小限制 | **1M 以内** |
| 预览视频 | ≤10M，封面图 1080×1920 或 1080×2340 |

### 1.2 命名规范

- 不超过 **10 个中文字**（含标点、空格）
- 须符合"**信、达、雅**"，简洁文雅，不可有歧义
- **禁止**前后空格，中间最多一个空格
- **禁止**特殊符号、敏感词汇
- 不可与官方资源名称雷同
- 不可包含第三方广告信息、网址、产品名
- **禁止**功能描述词（充电、流体）、系统名（OS、Win、鸿蒙）
- 同类型名称不能仅以数字（XX1、XX2）或时间（正午、傍晚）区分

### 1.3 标签规范

分 **4 个维度** 勾选，不标注同义词、无意义标签、与画面无关的标签：
- **画面标签**：图片主体内容描述
- **颜色标签**：主色调
- **气氛标签**：画面传达的情绪氛围
- **风格标签**：艺术风格分类

### 1.4 内容图片上传规则

- 猜谜/emoji/成语/明星小时候/找不同/谐音梗/脑筋急转弯等使用**大面积画框修饰**的插画图片，每月上传比例暂调整为当月审核上限的 **5%**
- Emoji猜谜专栏送审数量占比不得高于当月送审总图数的 **10%**（2月起执行）

## 二、内容红线清单

### 2.1 政治/意识形态（一票否决）

- 不得含有意识形态元素
- 不得出现各国国旗、国徽、地图（含可分辨国家地区轮廓的地球设计）
- 小熊维尼动画形象不得出现
- 不得恶搞、篡改、歪曲历史人物和事件
- 不得出现公职人员形象（包括交警等）
- 国际局势紧张期间，涉及**日本元素**（动漫/影视/旅游/医疗等）谨慎上传
- "琉球"表述须删除"日本"字样，改为"南西诸岛"或直接删除
- 金色边瓶子 + "皇帝"文案组合需谨慎，避免恶意解读

### 2.2 色情/低俗

- 禁止淫秽色情、软色情、性暗示
- 禁止大面积肉色透视装、胸沟明显、露肚脐
- 禁止兜裆布等不雅着装
- 眉钉、舌钉、脐钉等非主流身体装饰不推
- 纹身禁止出现

### 2.3 暴力/恐怖/惊悚

- 禁止血腥画面、打架斗殴、骷髅、动物骨架/干尸
- 禁止画面氛围诡异惊悚（用户打开锁屏可能吓一跳）
- 禁止表情痛苦、引人不适的内容
- 画面过于压抑、看着心酸难受的不上

### 2.4 宗教迷信

- 禁止所有宗教人物形象（观音、佛像、耶稣、圣母玛利亚、穆罕默德等一切宗教画像/雕塑）
- 禁止阴阳八卦、占卜算命、风水罗盘
- 禁止符纸
- 禁止涉及死亡相关（遗作、爱宠离世变钻石等）
- 禁止迷信内容（招财猫、风水气场等）
- 禁止宗教建筑作为画面主体（寺庙/教堂/清真寺等，如仅作为背景且不可辨识具体宗教属性可酌情）

### 2.5 品牌/商业

- 不得出现任何品牌 Logo、品牌名（含华为/HUAWEI/HONOR/荣耀）
- 不得出现系统标识（EMUI/HarmonyOS/鸿蒙）
- 无授权不得使用品牌车辆、品牌玩偶、品牌设备
- 禁止广告/软广，不得出现 APP 名称、具体品牌名
- **禁止出现任何游戏名称**（含手游/端游/主机游戏）
- 禁止出现真实公司名称、真实个人信息、手机号、发票等
- 避免出现第三方引擎字样（掌酷等）
- 禁止出现任何形式的金钱元素或货币符号

### 2.6 奥运/世界杯

- 公司营销要求，**避免出现奥运和世界杯内容**，包括"奥运""世界杯"字眼

### 2.7 飞机/航行安全

- 避免出现危害航行的内容（不论现实还是影视），如鸟撞飞机
- 避免飞机机身残破/坠机现场感

### 2.8 非主流婚恋观

- 不上架同性恋、未成年人恋爱、不伦恋、婚外情、一夜情、不婚不育思想等内容
- 相关动漫（涉及LGBTQ主题的）不推

### 2.9 明星相关

- 避免花边新闻、负面内容、卖惨
- 避免给明星打非"大众化"标签（容易引发粉丝投诉）
- 避免"阴阳怪气"式文案
- 避免出现"踩一捧一"的配图
- 避免涉及法制咖
- 未经官宣的信息不传
- 明星真实学校/经历要核实准确
- 游戏、影视、明星三类资源上架需统一评审

### 2.10 社会敏感

- 避免现实社会负面内容
- 避免反映其他国家负面内容
- 避免涉及社会经济现状（如油价上涨引发新增成本等）
- 避免暗讽品牌/行业
- 避免涉及宗教信仰、民族饮食习惯（如回民不吃猪肉）
- 避免出现国家"三有"保护动物的食用内容
- 避免存在争议且未经普遍证实的事件

### 2.11 更多禁止元素

- 禁止素人正脸图片
- 禁止引导操作的按钮、二维码、条形码、水印
- 禁止为图片增加边框
- 禁止动物形象作为广告画面主体
- 禁止画面出现真实手机号、联系人名称
- 禁止简介中出现福利、返现、赠送等诱导内容
- 禁止图片中出现可能引发联想的敏感数字组合（如64、89等）

## 二-A、谐音梗专项质量规范（🆕）

> 谐音梗是当前驳回最高频区域（5-6月占互动类驳回50%，跨月重复出现），必须建立专项质量标准。

### 谐音梗三原则

**原则一：同音原则 —— 必须同音同调**
- ✅ 正确示范：羊→洋（同音同调，积极寓意）、鹿→禄（同音同调，吉祥寓意）、鱼→余（同音同调）
- ❌ 错误示范：鸡→饥（不同调）、花→发（不同音）
- 审核标准：用标准普通话朗读，两字读音（含声调）必须完全一致

**原则二：无害原则 —— 不得含贬义/歧视/负面含义**
- ❌ 禁止：无人在意（贬义）、摆烂（消极）、社畜（负面自嘲）、躺平（消极）
- ❌ 禁止：任何涉及人身攻击、外貌羞辱、智商贬低的谐音
- 审核标准：去掉谐音外壳后，原词是否正面或中性？负面的一票否决

**原则三：非人原则 —— 不得使用真实人名**
- ❌ 禁止：苏轼→舒适圈、鲁迅→鲁讯、李白→李拜
- ❌ 禁止：任何在世/已故公众人物的姓名谐音
- ❌ 禁止：明星/艺人/政治人物的姓名谐音
- 例外：虚构角色名（如动漫人物）可酌情，但需避免商业化IP

### 谐音梗内容提交自检
1. ☐ 谐音是否同音同调（用普通话朗读验证）？
2. ☐ 去掉谐音后原词是否正面/中性？
3. ☐ 是否涉及任何真实人名？
4. ☐ 是否涉及贬义/歧视/低俗？
5. ☐ 答案是否在落地页完整揭晓并做解释？

5项全部通过才可提交。

## 三、低质量内容判定标准

### 3.1 技术质量

- 存在噪点、马赛克、清晰度低、高曝光等问题
- 图片比例失调、界面不完整
- 文字过小、展示不完整、影响排版
- 预览视频拉伸变形、画面跳动闪烁、首尾不衔接

### 3.2 构图/审美

- 阴阳脸、主体物没有美感
- 画面简陋、元素质感廉价、完成度低
- 背景显脏、海水显脏、色调灰暗
- 纹理感过重、画风老气、画风低幼
- 画面像"页游感"、人脸质感粗糙
- 玻璃痕迹像屏幕有划痕
- 人物主体密集且杂乱
- 背景过亮与人物分不开
- 秃头不美观
- 食物油腻感过重
- 图片下方有色块造成锁屏切割感

### 3.3 模板/排版

- 模板图片中红框标题文字需小于黄框版式文字
- 字太多太密、费眼睛
- 同一模板未经二次设计不得直接用于多套资源
- 同一作品各元素风格应保持一致
- 不可使用"说明书式"预览图
- 不可制作"带手机"样式的封面图
- 模板落地页不能过于简单（如只写答案不解释梗）

### 3.4 传统/老气元素（禁止）

- 建筑、街景、书桌仅搭配昼夜切换
- 贴纸、对联、窗花、灯笼、烟花、鞭炮
- 流水、瀑布、凉亭、群山、荷塘、锦鲤、鹤、鹰等传统元素堆砌
- 蓝天、白云、草地、郁金香、彩霞等单调画面
- 海浪、海、沙滩、岩石等无功能性交互
- 星空、星云、星球、黑洞等合成图无创意动效
- 点线面抽象画面无功能性交互

## 五、品类流量层级与CTR规律（v4.0 重写）

> 数据来源：2026年1-5月32,270条单品级后台数据 + 7期月报(2025.09~2026.04) + 2期5月月报

### 5.1 核心概念：品类流量层级

平台对不同品类的流量分配存在显著差异。**品类内容量多≠供给过剩，而是品类流量池大+正反馈循环的结果**。不同品类不能跨品类比较绝对曝光量——应在同一品类内部看"最好内容"和"平均水平"的差距（操作空间）。

### 5.2 ██ 第一层：流量大品类（平台主力推送，品类内竞争是主战场）

| 品类 | 内容量 | 总曝光 | 平均曝光/条 | 平均CTR | 品类内CTR天花板 | 天花板案例 | 操作空间 |
|------|--------|--------|-----------|---------|----------------|-----------|----------|
| 动物萌宠 | 6640 | 14.9亿 | 22.5万 | 4.08% | 12.87% | 动物的奇特睡姿 | **+8.79pp** |
| 旅行风光 | 1877 | 0.66亿 | 3.5万 | 4.26% | 11.55% | 超现实照片 | **+7.29pp** |
| 明星 | 1292 | 1.52亿 | 11.7万 | 6.93% | 24.54% | 从小帅到大的男星 | **+17.61pp** |

**策略**：这些品类的主战场在品类内部——比拼的不是"做不做"，而是"比别人做得好多少"。
- 动物萌宠：砍掉"泛猫狗图集"（CTR仅3-4%），聚焦猎奇切口（睡姿/行为/冷知识，CTR 12%+）
- 旅行风光：砍掉"普通风光大片"，只做"超现实""像P过""不像真的"猎奇视角
- 明星：主攻互动猜谜型(CTR 22-25%)和颜值成长盘点型(CTR 24.54%)

### 5.3 ██ 第二层：流量中品类（有稳定流量池，用效率弥补体量）

| 品类 | 内容量 | 总曝光 | 平均曝光/条 | 平均CTR | 品类内CTR天花板 | 天花板案例 | 操作空间 |
|------|--------|--------|-----------|---------|----------------|-----------|----------|
| 插画 | 359 | 1.04亿 | 29.0万 | 15.63% | 19.58% | 谐音梗猜成语 | **+3.95pp** |
| 座驾 | 1090 | 1.90亿 | 17.4万 | 3.28% | 8.95% | 现实版《飞驰人生》 | **+5.67pp** |
| 动漫 | 235 | 0.26亿 | 11.0万 | 6.98% | — | — | — |
| 体育 | 1092 | 0.50亿 | 4.5万 | 4.32% | 9.43% | 地毯上的芭蕾 | **+5.11pp** |

**策略**：精选高CTR公式，用效率弥补流量池的体量差距。
- 插画：互动挑战(猜谜/测试/考考你) + 心理测试/育儿知识
- 座驾：问句痛点型("怎么办？")+ 热门影视IP绑定
- 动漫：国漫IP冷知识/彩蛋 + 新电影幕后，暑期档重点发力

### 5.4 ██ 第三层：流量小品类（平台分配曝光总量有限，做精品不做量）

| 品类 | 内容量 | 总曝光 | 平均曝光/条 | 平均CTR | 品类内CTR天花板 | 天花板案例 | 操作空间 |
|------|--------|--------|-----------|---------|----------------|-----------|----------|
| 科技 | 396 | 2.20亿 | 55.5万 | 3.43% | 30.95% | 黑洞也会"挑食"？ | **+27.52pp** |
| 艺术 | 2408 | 5.61亿 | 23.3万 | 3.42% | 26.10% | 白玉透雕孔雀衔花纹佩 | **+22.68pp** |
| 美食 | 463 | 0.12亿 | 2.5万 | 12.48% | — | — | — |
| 影视 | 233 | 0.06亿 | 2.8万 | 5.09% | — | — | — |
| 家居 | 154 | 0.03亿 | 1.8万 | 8.24% | — | — | — |

**策略**：品类总曝光池有限，每条内容都要走高CTR路线。做精品不做量。
- 科技：品类内CTR天花板极高(30.95%)，问句模式+猎奇反常识为主
- 艺术：聚焦文物命名型(CTR 15-26%)，不做泛"搞笑图集/视错觉"
- 美食：CTR高但曝光基数低，保持现有节奏

> ⚠️ 解读注意："平均曝光/条"是品类总曝光÷内容量。科技55.5万/条高是因为内容量极少(396条)，**不代表该品类流量大**。
> 正确用法：看"品类内CTR天花板"列——同一品类里最好内容和平均水平的差距，这才是可操作空间。

### 5.5 供给结构洞察（修正版）

```
流量大品类 + 品类内操作空间大 → 动物萌宠(+8.79pp)、明星(+17.61pp)、旅行风光(+7.29pp) → 精选切口提质
流量大品类 + CTR绝对水准高     → 动物萌宠(泛图集4%→猎奇切口12%+) → 降量提质
```

## 六-A、副标题（锁屏文案）句式公式库

> 来源：V2.0深度分析手册——376条优秀案例副标题的逐条归纳。副标题≤25字，核心角色是「把标题抛出的钩子接住并放大」。

### 通用型公式（公式1-4）

**公式1：主体+动感描述，+意外细节/情绪爆发**
```
适用：几乎全部品类，最通用安全
示例：
→ "狗狗美容变毁容！网友：404错误，狗不见了！"
→ "不是特效！米兰街头惊现拉链房子，夜晚还会变蓝发光！"
→ "船像在没有水的半空中行驶一般！中国有一地上榜。"
```

**公式2：从X到Y，+ 个个/每个/每一X都Z**
```
适用：盘点型标题的标配，动物、明星品类
示例：
→ "从蝙蝠侠到刘亦菲，这只猫咪跨界碰瓷娱乐圈，相似度爆表！"
→ "盘点十大动物成精瞬间，每一个都在跟摄影师抢饭碗！"
```

**公式3：你V过吗？+ 答案/反转**
```
适用：动物、旅行、互动类，高互动率品类首选
示例：
→ "不信你看看这组图！最后一张笑翻网友。"
→ "你敢住吗？其中一座就在中国贵州！"
```

**公式4：不是A，而是B / 原来是C**
```
适用：科技、文化、生活，反差颠覆型标题的完美搭配
示例：
→ "不是特效！米兰街头惊现拉链房子，夜晚还会变蓝发光！"
→ "表面是当红明星，私下却身怀硬核技能，个个深藏不露。"
```

### 品类特化型公式（公式5-8）

**公式5（明星专属）：人名+作品/角色/关系 + 情感标签 + 热议指标**
```
示例：
→ "黄子韬与徐艺洋正式公开婚纱照，其中多组画面引发热议，浪漫氛围拉满！"
→ "从团队'拖油瓶'到光之晨曦战力核心，陈樱儿神眷觉醒实力大涨？"
```

**公式6（动物专属）：动物+拟人化行为+情绪化感叹，+ 读者反应预测**
```
示例：
→ "猫咪哀嚎：我成沙发附属品了！主人已被逼疯！"
→ "狗的7个'自恋'证据，别怀疑，它可能沉浸在自己的帅气里无法自拔！"
```

**公式7（科技专属）：科学事实+震撼量化数据，+ 对人类/地球的影响延伸**
```
示例：
→ "质量达225个太阳，刷新纪录！已经133亿岁高龄。"
→ "从被质疑的'鬼魅'现象，到改变世界的核心技术，量子纠缠只用了不到一百年！"
```

**公式8（互动猜谜专属）：规则说明，+ 挑战难度预告 + 身份标签诱惑**
```
示例：
→ "Emoji化身明星密码，脑洞越大越易猜，看看你是他的粉丝吗？"
→ "黑色轮廓藏着我们熟知的动漫角色，能猜出来的都是真正懂动漫的人！"
```

### 感染力强化型公式（公式9-12）

**公式9：感叹前置——「太X了！/ 不敢信！/ 万万想不到！」**
```
把最强情绪词放句首，一秒锁定注意力。
```

**公式10：后果预告——「否则/小心/千万别……」**
```
制造轻微焦虑感驱动点击，适用于生活科普和安全警示类。
```

**公式11：身份标签诱惑——「X的人才是真Y」「能X的不超过Z%」**
```
给读者一个"我属于某个独特群体"的身份认同动机。
示例：
→ "能猜出6个以上的绝对是王者段位！"
→ "全吃过的堪称资深吃货！"
```

**公式12：进度锚点——「最后一张/最后一页有惊喜」**
```
在副标题中暗示结尾有彩蛋，驱动完读率。
示例：
→ "看完别笑！最后一个直接让人笑到窒息。"
```

### 11.3 源文章筛选检查清单

拿到一篇源文章后，逐条确认：
1. ☐ 红线过滤：无政治/宗教/色情/暴力/负面/品牌广告内容？
2. ☐ 品类适配：文章角度是否符合该品类的锁屏特征规律？
3. ☐ "哇"瞬间：有没有一个具体细节/数据/事实能驱动好奇心？
4. ☐ 图片可行：这个角度能找到高质量图片素材吗？
5. ☐ 转换可行：能被压缩为≤10字标题+≤25字副标题+150-300字落地页吗？

5条全部通过才可进入选题生成。任意1条不通过→放弃这篇源文章，换下一篇。
