# 华为画报内容策划助手 (huabao-content-planner)

华为杂志锁屏运营专用 AI Skill，覆盖 13 个内容品类。融合**选题三层分析法**（定大品类→找优势方向→精细切分）和**三个运营思维**（保持优势→观察同行→尝试突破），从 330+ 条优秀案例中提炼标题六种模式+副标题四法则，结合 CTR 数据、22 条审核红线、真实驳回案例库，提供选题策划→文案生成→红线自检一站式能力。

## 版本

v3.0.0 (2026-06-01)

## 核心方法论

### 选题三层分析法

```
大品类定位 → 优势方向锁定 → 精细切口拆解
（数据驱动）   （品类内子方向）   （同类翻新角度）
```

- **第一层**：从 CTR 数据出发，确定当前优先发力的品类
- **第二层**：在品类内找到数据最好的子方向（如萌宠→猫狗搞怪反差冷知识、明星→一二线造型冷知识）
- **第三层**：在优势方向内不断翻出新角度，避免同质化

### 三个运营思维

| 思维 | 说明 |
|------|------|
| **保持优势** | 数据验证有效的策略、栏目、风格稳住不丢 |
| **观察同行** | 每天刷样机推送，学习竞品标题策略（如视觉中国"动词+反问"公式） |
| **尝试突破** | 前瞻性地做新形式、新栏目、新风格（轻互动新模板等） |

### 标题六种高效模式

| 模式 | 特征 | 示例 |
|------|------|------|
| 问句悬念 | 抛出让人想知道答案的问题 | "你能看到什么形状？" |
| 猎奇反常识 | 打破日常认知的事实 | "长得像AI的动物" |
| 数字盘点 | 具体数字传递信息密度 | "全球五大奇险建筑" |
| 警示别xxx | "千万别"触发自我保护点击 | "千万别这样安慰人" |
| 引号梗 | 引号制造网感和默契 | ""吃同事"的大鱼" |
| 身份代入 | 标题与"你"产生关联 | "学霸是这样养成的" |

### 副标题四法则

1. **接力不重复** — 标题抛悬念，副标题给具体事实钩子
2. **口语化** — 像朋友聊天，"？""！""……"
3. **"你"字高频** — 把读者拉入对话
4. **句末留余味** — 结尾让人想点进去

### 关键数据洞察

- **互动=收入之王**：14 组 10w+收入中 10 组为轻互动型，明星童年照最高 40w+
- **增产验证**：插画 283→2023 条，CTR 从 11.95%→13.86%（量增CTR反升）
- **品类供给失衡**：动物 10534 条+旅行风光 11721 条占总量 47% 但 CTR 仅 5.8%，存在巨大提质空间

## 安装方式

### Proma / Claude Code
将 `skills/` 放入工作区 skills 路径，`workspace-files/.context/` 放入上下文路径。

### 通用 AI 工具
打开 `workspace-files/huabao-content-planner-package/standalone-prompt.md` 粘贴到系统提示词。
