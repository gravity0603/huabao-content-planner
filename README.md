# 华为画报内容策划助手 (huabao-content-planner)

华为杂志锁屏运营专用 AI Skill，覆盖 13 个内容品类。从 330+ 条月报优秀案例中提炼出**标题六种高效模式**和**副标题四法则**，结合 CTR 数据、30+ 条审核红线和真实驳回案例库，提供选题策划、文案生成、红线自检一站式内容创作能力。

## 核心能力

### 策略反推
每次策划前先拆解月报优秀案例的底层策略——标题用什么句式、副标题的钩子在哪、图片选什么主体，抽取出可复用模式再生成内容。

### 标题六种高效模式

| 模式 | 特征 | 示例 |
|------|------|------|
| 问句悬念 | 抛出让人想知道答案的问题 | "你能看到什么形状？" |
| 猎奇反常识 | 打破日常认知的事实 | "长得像AI的动物" |
| 数字盘点 | 具体数字传递信息密度 | "全球五大奇险建筑" |
| 警示别xxx | "千万别"触发自我保护点击 | "千万别这样安慰人" |
| 引号梗 | 引号制造网感和默契 | ""吃同事"的大鱼" |
| 身份代入 | 标题与"你"产生关联 | "学霸是这样养成的" |

### 副标题四法则

1. **接力不重复** — 标题抛悬念，副标题给具体事实钩子
2. **口语化** — 像朋友聊天，"？""！""……"
3. **"你"字高频** — 把读者拉入对话
4. **句末留余味** — 结尾让人想点进去

### 红线自检+驳回对照
20 条红线逐条确认，涵盖政治/色情/暴力/宗教/品牌/奥运/军字/明星舆情/绝对性表述等，对照月报真实驳回案例主动规避。

### 批量选题生成
输出格式：标题（≤10字）+ 副标题（≤25字）+ 落地页文案 + 策略反推 + 红线自检。强制计数标注，超字即改。

### 13品类支持
插画 / 美食 / 家居 / 科技 / 动漫 / 动物 / 明星 / 旅行风光 / 影视 / 座驾 / 艺术 / 体育 / 花卉绿植

## 安装方式

### Proma / Claude Code
将 `skills/` 目录放入工作区 skills 路径，`workspace-files/.context/` 放入上下文路径。

### 通用 AI 工具（GPT / Dify / Coze / 豆包 / Kimi 等）
打开 `workspace-files/huabao-content-planner-package/standalone-prompt.md`，复制粘贴到系统提示词即用。

## 使用方法

```
给我10个{品类}选题，侧重{方向}
```
- `给我10个明星类选题，侧重盘点反差`
- `给我5个插画类选题，侧重互动猜谜`
- `检查这个选题：标题"xxx" 副标题"xxx"`
