# 华为画报内容策划助手 (huabao-content-planner)

华为杂志锁屏运营专用 AI Skill，覆盖 13 个内容品类，提供选题策划、文案生成、红线自检一站式内容创作能力。

## 版本

v2.0.2 (2026-05-25)

## 文件结构

```
├── skills/huabao-content-planner/
│   └── SKILL.md              # Skill 定义（角色、工作流、红线清单、输出模板）
├── knowledge-base/
│   └── huabao-knowledge-base.md  # 知识库（CTR数据、内容公式、驳回案例、专栏清单）
└── README.md
```

## 功能概述

- **批量选题策划**：13 品类 × 策略反推 × 热点驱动，生成标题+锁屏文案+落地页文案
- **红线自检**：19 条红线逐条确认，覆盖政治/色情/暴力/宗教/品牌/军字/绝对性表述等
- **数据驱动**：基于真实 CTR 数据（单品级 6604 条）的策略反推
- **知识库可更新**：新材料追加即生效，版本记录完整

## 使用方法

在支持 Claude Agent SDK 的平台（如 Proma）中：

1. 将 `skills/huabao-content-planner/SKILL.md` 放入 skills 目录
2. 将 `knowledge-base/huabao-knowledge-base.md` 放入项目上下文路径
3. 调用：`/huabao-content-planner 给我5个美食类选题`

## 数据来源

- 华为开发者官网《杂志锁屏素材审核规范》
- 7 期供应商月报 (2025.09~2026.04)
- 内容核心指标明细表 (6604 条单品级数据, 2026.05.21)
