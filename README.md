# huabao-content-planner

华为杂志锁屏内容策划助手 Skill。从月报优秀案例反推创作策略，基于六种标题模式+副标题四法则生成选题。

## 版本历史

### v4.3.0 (2026-06-25)

**新增规则：**
- 🔴 标题语序规范：中文标题必须为名词短语/直接问句/反常识断言，禁止机器翻译语序
- 🔴 字数精确计数规则：含所有标点符号，输出时逐字列出计数过程
- 🔴 素材库优先·强制执行：选题策划必须先走素材库源账号，禁止跳过直接WebSearch
- 🔴 输出前日报表/已做CSV逐条查重：每条标题与用户已上传历史逐条比对
- 座驾品类方向修正：座驾=生活化驾驶冷知识，禁止车史/赛车/品牌史

**更新项：**
- 出街前强制自查：新增⑤日报表查重、⑥标题朗读测试
- 高频错误清单：新增⑦字数不含标点、⑧标题语序、⑨未查重、⑩座驾方向偏离
- 硬拦截协议：新增⑧日报表查重、⑨字数计数错误、⑩座驾品类偏离
- 自检报告模板：新增日报表查重、字数计数、语序、座驾方向、素材库执行等检查项

### v4.2.1 (2026-06-20)

- 新增标题人名禁令（v4.2.1）：所有品类标题一律不得出现具体人物姓名

- 硬拦截协议新增⑥标题人名红线、⑦标题术语红线
