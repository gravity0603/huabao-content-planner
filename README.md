# 华为画报内容策划助手 (huabao-content-planner)

华为杂志锁屏运营专用 AI Skill，覆盖 13 个内容品类。融合**素材库优先检索** + **品类流量层级** + **数据驱动方法论** + **源文章拆解五步法** + **13品类特征规律** + **选题三层分析法** + **三个运营思维**，从 32,270 条真实后台数据中量化验证标题模式效率排名 P0-P5，内置 40+ 验证源账号素材库，提供选题策划→文案生成→红线自检一站式能力。

## 版本

v4.2.0 (2026-06-10)

## 核心方法论

### 场景碰撞法（v4.2 新增）

标题里必须有至少一个词来自非本品类的日常场景。好标题 = 句式 × 场景碰撞。

✅ "有钻石的沙滩" ‖ ❌ "峡谷窄到0.4米"

| 品类 | 碰撞方向 | 示例 |
|------|---------|------|
| 旅行 | 购物词/身体动作 + 地理 | 有钻石的沙滩、侧身才能过的峡谷 |
| 动物 | 影视词/社交词 + 动物 | 动物世界名场面、跨物种友谊 |
| 明星 | 游戏词/弹幕词 + 明星 | 中单男人大赏、关掉美颜的女星 |
| 穿搭 | 劝退词 + 具体单品 | 不建议买的衣服颜色 |
| 美食 | 吐槽词 + 食物 | 反人类的美食设计 |

### 信息源标注规范（v4.2 新增）

每条选题信息源须诚实标注验证状态：`[已验证]` / `[待验证]` / 无源。**禁止伪造链接**，弄虚作假一票否决。

### 检索策略分层（v4.2 更新）

素材库 → 权威媒体 → 商业平台 → 如实告知。前两层不足时，不硬凑不伪造。

### 素材库优先检索（v4.1 新增）

选题素材检索遵循**素材库→同类扩散→全网补位**三层漏斗，替代散乱的多平台泛搜。按品类从 40+ 验证源账号定点扫货，源文章须经过红线过滤+品类适配+"哇"瞬间提取后才可转化为选题。

### 数据驱动 & 品类流量层级（v4.0 新增）

**北极星指标**：品类内综合得分 = 曝光次数 × CTR。不同品类不能跨品类比较曝光量——平台对每个品类的流量分配有显著差异。

| 层级 | 品类 | 策略 |
|------|------|------|
| **第一层** 流量大品类 | 动物萌宠、旅行风光、明星 | 品类内竞争是主战场，拼"比别人好多少" |
| **第二层** 流量中品类 | 插画、座驾、动漫、体育 | 用CTR效率弥补流量池体量差距 |
| **第三层** 流量小品类 | 科技、艺术、美食、影视 | 做精品不做量，每条走高CTR路线 |

### 标题模式效率排名（32,270条数据量化）

| 优先级 | 模式 | 平均CTR | 批次占比 | 最佳适用 |
|--------|------|---------|----------|----------|
| P0 | 问句悬念（"？"） | 16.28% | ≥40% | 全品类通用 |
| P1 | 引号梗（"xx"） | 17.50% | ≥20% | 动物/明星/科技 |
| P2 | 猎奇反常识 | 14.91% | ≥15% | 科技/旅行 |
| P3 | 文物命名 | 15.38% | 艺术专属 | 艺术 |
| P4 | 身份代入（"你"） | 13.81% | ≥10% | 插画/育儿 |
| P5 | 数字盘点 | 16.05% | ≤10% | 口味调剂 |

### 双重钩子组合（v4.0 新增）

问句+引号 = 最优组合（示例：`黑洞也会"挑食"？` CTR 30.95%）

### 月报与数据双轮驱动（v4.0 新增）

后台数据告诉你"用户在点什么"，月报告诉你"平台在推什么"。每月收到月报后交叉对比，交集就是最优策略。

## 安装方式

### Proma / Claude Code
将 `skills/` 放入工作区 skills 路径，`workspace-files/.context/` 放入上下文路径。

### 通用 AI 工具
打开 `workspace-files/huabao-content-planner-package/standalone-prompt.md` 粘贴到系统提示词。

## 使用方法

```
给我10个{品类}选题，侧重{方向}
```
