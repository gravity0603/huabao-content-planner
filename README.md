# 华为画报内容策划助手 (huabao-content-planner)

华为杂志锁屏运营专用 AI Skill，覆盖 13 个内容品类。融合**源文章拆解五步法** + **13品类特征规律** + **选题三层分析法** + **三个运营思维**，从 330+ 条优秀案例中提炼标题六种模式+副标题四法则，结合 CTR 数据、22 条审核红线、真实驳回案例库，提供选题策划→文案生成→红线自检一站式能力。

## 版本

v3.3.0 (2026-06-03)

## 核心方法论

### 源文章拆解五步法

```
锁定源文章富矿 → 拆解"源→锁屏"转换 → 同账号扫货 → 同类型账号扩散 → 验证品类特征
```

优秀案例不是凭空创作的——每条都有源文章。五步法从源头解决"选题从哪来"的问题。

### 13品类特征规律

| 品类 | 特征 | 核心操作 |
|------|------|---------|
| 动物 | 猫狗有趣/搞怪/冷知识 | **人格化叙事**——给行为贴人设标签 |
| 明星 | 造型/身材/风格/穿搭/妆容 | **情绪翻转正**——纯造型，不涉角色分析 |
| 旅行 | 猎奇/有趣/冷知识 | **知识降维**——长文知识→短句悬念 |
| 插画 | 自我验证冲动 | 把"阅读"变成"参与"（测测你/考考你） |
| 影视 | 热播剧单一细节 | 收缩焦点到极致，不做大盘点 |
| 科技 | 宇宙/天文猎奇 | **拟人化翻译**——黑洞"干饭" |
| 动漫 | 圈内人密码 | IP粉丝才懂的彩蛋/冷知识 |
| 艺术 | 文物→人话 | 反差趣味或命名美学 |
| 美食 | "被骗了"纠正冲动 | 你一直以为A，其实答案是B |
| 体育 | 反差命名 | 用优雅词汇重新命名运动项目 |
| 座驾 | 绑定生活场景 | 讲开车困惑，不讲超跑参数 |

### 选题三层分析法

```
大品类定位 → 优势方向锁定 → 精细切口拆解
```

### 三个运营思维

| 思维 | 说明 |
|------|------|
| **保持优势** | 数据验证有效的策略稳住不丢 |
| **观察同行** | 刷样机推送，学竞品标题策略（视觉中国"动词+反问"） |
| **尝试突破** | 前瞻性做新形式、新栏目（轻互动新模板等） |

### 标题六种模式 & 副标题四法则

| 模式 | 示例 |
|------|------|
| 问句悬念 | "你能看到什么形状？" |
| 猎奇反常识 | "长得像AI的动物" |
| 数字盘点 | "全球五大奇险建筑" |
| 警示别xxx | "千万别这样安慰人" |
| 引号梗 | ""吃同事"的大鱼" |
| 身份代入 | "学霸是这样养成的" |

### 高频错误清单

1. 与优秀案例库重复——禁止直接搬用
2. 与对话历史重复——每批生成前回顾
3. 明星品类脱轨——纯造型穿搭，禁滑向角色分析
4. 标题/副标题语病——输出前朗读一遍

### 关键数据

- **互动=收入之王**：14组10w+收入中10组为轻互动，明星童年照最高40w+
- **增产验证**：插画283→2023条，CTR从11.95%→13.86%（量增CTR反升）

## 安装方式

### Proma / Claude Code
将 `skills/` 放入工作区 skills 路径，`workspace-files/.context/` 放入上下文路径。

### 通用 AI 工具
打开 `workspace-files/huabao-content-planner-package/standalone-prompt.md` 粘贴到系统提示词。

```
给我10个{品类}选题，侧重{方向}
```
