# 华为画报内容策划助手 (huabao-content-planner)

华为杂志锁屏运营专用 AI Skill，覆盖 13 个内容品类。融合**数据驱动标题体系** + **副标题12套公式库** + **选题季节节律** + **谐音梗三原则** + **互动类专项红线** + **热点感知流程** + 素材库优先检索 + 品类流量层级 + 13品类特征规律，从 32,270 条真实后台数据 + 110 条月报优秀案例 + 22 条实操驳回记录中提炼出可复用的选题方法论，提供选题策划→文案生成→红线自检一站式能力。

## 核心方法论

### 数据驱动标题体系（v5.0 重写）

**不做理论分类，以优秀案例和高CTR数据为唯一依据。** 从 3-4 月月报 110 条优秀案例 + 5.21 单品 CTR Top 数据中，按品类逐一提取真实高曝光标题，归纳可复用方向和参考模板。泛化标题（"可爱的猫""美丽的风景"）在优秀案例中**零出现**。

核心规律：「猜」和「问」两个字撑起了最高 CTR 的半壁江山。所有高 CTR 标题的共同特征是**让读者脑中产生一个问题**。

### 副标题句式公式库（v5.0 新增）

12 套可复用副标题句式公式，分三类：
- **通用型（公式1-4）**：主体+意外细节 / 从X到Y / 你V过吗+反转 / 不是A而是B
- **品类特化型（公式5-8）**：明星专属 / 动物专属 / 科技专属 / 互动猜谜专属
- **感染力强化型（公式9-12）**：感叹前置 / 后果预告 / 身份标签 / 进度锚点

内置标题方向 × 副标题公式匹配矩阵，避免每次从零写起。

### 选题季节节律图谱（v5.0 新增）

月度选题热度表 + 7月/10月 CTR 全年双高峰洞察 + 动物品类子主题季节轮换 + 当前月选题指引。关键洞察：7 月和 10 月选题筹备至少提前 2 周启动。

### 谐音梗三原则 + 互动类专项红线（v5.0 新增 · 驳回数据驱动）

基于 5-6 月 22 条实操驳回记录（互动类占 45%，谐音梗跨月重复出现），建立：
- **谐音梗三原则**：同音同调 + 无贬义 + 不用真实人名
- **11 条互动类专项红线**：人名梗禁止 / 英文标点禁止 / Emoji 一一对应 / 答案完整上传 / 心理测试逻辑链 等
- **动态驳回高亮区**：每月根据最新驳回数据刷新最高频雷区

**红线清单从 17 条扩展至 29 条**（通用 17 条 + 互动专项 12 条）。

### AI 写作指令模板（v5.0 新增）

5 套分品类 AI 指令模板（动物猎奇 / 明星轻互动 / 旅行猎奇 / 科技冷知识 / 互动猜谜），每套内置概念锚点 + 结构锚点 + 语气锚点。

### 热点选题策划流程（v5.0 新增）

6 步流程：热点获取 → 品类匹配 → 时效分级 → 数据驱动标题 → 红线扫描 → 输出方案。内置节日/影视/科技/双高峰的时效节奏参考。热点内容的 CTR 通常是常青内容的 2-3 倍。

### 素材库优先检索（v4.1）

选题素材检索遵循**素材库→同类扩散→全网补位**三层漏斗，替代散乱的多平台泛搜。按品类从 40+ 验证源账号定点扫货，源文章须经过红线过滤+品类适配+"哇"瞬间提取后才可转化为选题。

### 品类流量层级（v4.0）

| 层级 | 品类 | 策略 |
|------|------|------|
| **第一层** 流量大品类 | 动物萌宠、旅行风光、明星 | 品类内竞争是主战场 |
| **第二层** 流量中品类 | 插画、座驾、动漫、体育 | 用CTR效率弥补流量池差距 |
| **第三层** 流量小品类 | 科技、艺术、美食、影视 | 做精品不做量 |

### 场景碰撞法（v4.2）

标题里必须有至少一个词来自非本品类的日常场景。好标题 = 句式 × 场景碰撞。

### 标题模式效率排名（32,270条数据量化）

| 优先级 | 模式 | 平均CTR | 批次占比 | 最佳适用 |
|--------|------|---------|----------|----------|
| P0 | 问句悬念（"？"） | 16.28% | ≥40% | 全品类通用 |
| P1 | 引号梗（"xx"） | 17.50% | ≥20% | 动物/明星/科技 |
| P2 | 猎奇反常识 | 14.91% | ≥15% | 科技/旅行 |
| P3 | 文物命名 | 15.38% | 艺术专属 | 艺术 |
| P4 | 身份代入（"你"） | 13.81% | ≥10% | 插画/育儿 |
| P5 | 数字盘点 | 16.05% | ≤10% | 口味调剂 |

## 安装方式

### Proma / Claude Code
将 `skills/` 放入工作区 skills 路径，`workspace-files/.context/` 放入上下文路径。

### 通用 AI 工具
打开 `workspace-files/huabao-content-planner-package/standalone-prompt.md` 粘贴到系统提示词。

## 使用方法

```
给我10个{品类}选题，侧重{方向}
检查这个选题：标题xxx 文案xxx
更新知识库，我有一份新的月报
策划6月热点选题
```
